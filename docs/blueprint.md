# **App Name**: LendLink

## Core Features:

- User Authentication: User registration and authentication.
- User Profiles: Profile creation for lenders and borrowers.
- Loan Listings: Loan listing and browsing interface.
- Loan Requests: Secure platform for users to request loans from each other.
- Risk Assessment: Use of a generative AI tool that estimates risk based on applicant-provided details, public data and platform usage data.
- Investment Tracking: Dashboard for lenders to track investments and returns.
- Notifications: Automated email notifications for loan updates, payments, and platform announcements.

## Style Guidelines:

- Primary color: Warm and inviting light-purple (#C2A3CE), fostering trust and security.
- Background color: Soft, desaturated light-purple background (#F1EFF2).
- Accent color: Blue-purple (#7B61FF), to highlight calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif offering a modern and neutral look for readability and clear communication.
- Consistent use of financial and security-themed icons.
- Clean and intuitive layout with clear sections for loan listings, user profiles, and investment tracking.
- Subtle animations for loading and transaction confirmations to provide feedback and enhance the user experience.
